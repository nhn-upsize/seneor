■ 환경 설정 방법 (스크립트 실행 시)
================================================================================

  ※ 엑셀/PPT/MD 파일만 보는 경우 설정 불필요

  1) Python 3.10 이상 설치
     https://www.python.org/downloads/

  2) 패키지 설치
     터미널에서 프로젝트 폴더로 이동 후:

     pip install -r requirements.txt

  3) .env 파일 생성 (API 호출이 필요한 경우만)
     프로젝트 루트에 .env 파일을 만들고 아래 내용 입력:

     SENSOR_TOWER_API_TOKEN=여기에_토큰_입력

     ※ 토큰은 보안상 zip에 포함하지 않았습니다
     ※ .cache/ 폴더가 있으면 기존 캐시로 결과 재현 가능 (토큰 불필요)
     ※ 새로운 기간/국가 데이터 수집 시에만 토큰 필요

■ 분석 기준 (중요!)
================================================================================

  1. 신규진입 정의
     - 각 연도 1월 1일 TOP 100을 기준선으로 설정
     - 2월~12월 중 처음 TOP 100에 진입한 게임 = 신규진입
     - 매년 1월은 기준선이므로 집계에서 제외

  2. 분석 기간
     - pre2025: 2022~2024년 각 2~12월 (총 33개월)
     - post2025: 2025년 2~12월 + 2026년 2~3월 (총 13개월)

  3. 3개월 생존
     - 첫 진입월 + 3개월 시점에 TOP 100에 존재하는지 여부
     - 예: 2025-04 진입 → 2025-07에 TOP 100 존재 = 생존

  4. 차트 잔존율
     - 신규진입 게임이 M+N 시점에 매출 TOP 100에 남아있는 비율
     - M+1, M+2, M+3, M+6, M+12 시점 확인
     - post2025의 M+6은 61%, M+12는 21%만 판정 기간 충분 (데이터 끝: 2026.03)

  5. 수치 기준
     - n (게임수): 월평균 (총 게임수 ÷ 해당 기간 개월수)
     - 생존율: 전체 기간 누적 비율 (총 생존수 ÷ 총 신규진입수)
     - 광고집행율: 월별 비율의 평균 (iOS만, AOS는 Sensor Tower 커버리지 한계)
     - 매출: 공식 App Store/Google Play 순위 기준, Sensor Tower 추정 매출

  6. 퍼블리셔 분류
     - 한국: South Korea
     - 일본: Japan
     - 북미: US, Canada
     - 중화권: China, Hong Kong, Taiwan, Macau
     - 기타: Finland, Singapore, UK, Turkey, Israel, Switzerland 등
       ※ 기타에 Supercell(핀란드), King(스웨덴) 등 글로벌 강자 포함

  7. 랭킹 기준일
     - 매월 1일 시점의 TOP 100 랭킹 사용

  8. 소표본 주의
     - n < 50인 경우 참고 수준으로 해석

■ 센서타워 사이트에서 데이터 검증 방법
================================================================================

  1. 상위차트 접속: https://app.sensortower.com/top-charts

  2. 설정:
     - 스토어: App Store (iOS) 또는 Google Play (AOS)
     - 기기: iPhone
     - 국가: 대한민국 / 일본 / 미국
     - 카테고리: 게임
     - 날짜: 확인하고 싶은 월의 1일 (예: 2025/01/01)

  3. 매출 순위 탭에서 순위 확인

  4. 신규진입 검증:
     - 해당 월과 이전 월 데이터를 비교
     - 이전 월에 없고 해당 월에 있는 게임 = 신규진입

  5. 검증 완료 예시:
     - JP AOS 2022-02 신규진입: 24개 ← 사이트와 정확히 일치 확인됨
     - KR iOS 2025-04 신규진입: 11개 ← 사이트와 정확히 일치 확인됨
